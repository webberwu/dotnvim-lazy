# dotnvim-lazy

## Installation

```shell
mv ~/.config/nvim{,.bak}
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}

ln -s <YOUR_REPO_PATH> ~/.config/nvim
```
